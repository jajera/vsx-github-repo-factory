# Change Log

All notable changes to the "GitHub Repo Factory" extension will be documented in this file.

## [0.1.0] - 2024-11-21

### Added

- Initial release
- Create GitHub repositories with optional template
- Create optional "First Release" issue with customizable title and description
- Create optional branch with issue number prefix (e.g., "1-first-release")
- Delete repository functionality with confirmation
- Modify repository settings (description, visibility, features, PR settings, etc.)
- Repository existence validation
- Branch name validation
- Workspace file creation (.code-workspace)
- Context directory setup (.demo-context/, .cursor/context.md)
- Progress indicators for all operations
- Error handling and user feedback
